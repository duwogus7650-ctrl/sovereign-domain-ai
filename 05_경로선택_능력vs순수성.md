# 개인이 만들 수 있는 소버린 AI: 경로와 선택

## 증류는 소버린의 *부품*이지 소버린 자체가 아니다 — 선생 출처가 가른다
1. 해외 폐쇄형(GPT·Gemini·Claude) 블랙박스 증류 → **이름뿐인 소버린**. 종속 + 약관 위반 위험 + 능력 천장(선생을 못 넘고 한국어 약점도 물려받음).
2. 해외 공개가중치(Llama·Qwen·GLM) 증류·추가학습 → **파생**. 라이선스 허용 시 합법이나 from-scratch는 아님.
3. 자체 from-scratch 모델 증류(self-distillation) → **진짜 소버린 스택**. 증류가 토대가 아니라 경량화·배포 도구로 들어감.

## 개인 스케일 4가지 경로 (노력·순수도 순)
1. 로컬 + RAG — 공개모델 그대로 + 내 문서 검색. 배포만 소버린.
2. **파인튜닝(LoRA/QLoRA)** — 공개모델에 도메인 데이터 주입. (현실적 80/20 시작점)
3. 도메인 추가 사전학습(continual pretraining) — 더 깊지만 여전히 파생.
4. from-scratch 소형 모델 — 0.5–3B 직접 학습. 100% 자체, 단 품질 제한.

## 프런티어 야심과 현실
"Opus 4.8급을 최대 노력으로" — 솔직한 현실: 프런티어 from-scratch는 개인 불가. **노력이 아니라 자본·규모의 문제.** 한 번 학습에 수만 GPU·수 개월·수억 달러·수백 명 조직. 국가급 독파모(5,300억·대기업·수천 GPU)도 목표가 '프런티어의 95%'이고 고전 중.

## 결정적 모순: 능력 vs 순수성은 양립 불가(개인 스케일)
- **능력 최대(A)**: 오픈웨이트 프런티어급(DeepSeek V4·Qwen3.5·GLM-5·Kimi K2.6)을 내 손으로 운영·특화. 능력 ↑, 단 '파생'. 얻는 소버린 = **소유·운영·통제**.
- **순수성 최대(B)**: from-scratch 소형(0.5–3B). 100% 자체, 단 능력 제한. 얻는 소버린 = **기원·창조**.
- 추가 아이러니: 지금 오픈웨이트 최상위는 대부분 중국 랩(DeepSeek·Moonshot·Zhipu·알리바바). '능력 우선'으로 가면 한국 소버린인데 중국 모델 위에 쌓게 됨(= 업스테이지 논란의 그 지점).

## "모방은 창조의 어머니" — 무엇을 모방하느냐가 정반대 결과
- **방법을 모방**(아키텍처·하이퍼파라미터·학습 레시피·데이터 구성법): 순수성 안 깨짐. 모든 from-scratch가 이미 이렇게 함. 출발선의 '시행착오'를 앞당겨줄 뿐, '능력 천장'을 올려주진 않음.
- **능력(가중치)을 모방**(복제 또는 증류): 능력 ↑, 그 순간 순수성 깨짐(정의상). '빌린 출발선'과 '순수성'은 동전의 양면. 증류한 학생은 선생을 못 넘어서 '모방 후 추월'도 불가.
- 비유: 화가 지망생이 거장의 '기법'을 흡수하는 건 순수성 유지(붓질은 끝까지 자기 손), 거장의 '완성작'을 오려 붙이는 건 영원히 남의 그림.

## 최종 선택
